## Transmisor Arduino

### ARDUINO  NRF24L01

- 8 		CSN
- 7 		CE
- 13 		SCK
- 11 		MOSI
- 12 		MISO


### Receptor NODEMCU

- D2       CSN
- D4       CE
- 3.3      V+
- GND      GND
- D7       MOSI 
- D6       MISO
- D5       CSK o SCK
